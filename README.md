# rhartlage.github.io

This repository is the root GitHub Pages landing page for published tools.

## Purpose

Use this site as a hub that links out to individual tool repositories such as:

- `https://rhartlage.github.io/adhoc-network-workbench/`
- `https://rhartlage.github.io/bus-2150-live/`

## Publish

1. Create a GitHub repository named `rhartlage.github.io`
2. Push the files in this folder to that repository
3. In GitHub, open `Settings -> Pages`
4. Deploy from the `main` branch and `/ (root)`

The site will then be available at:

- `https://rhartlage.github.io/`

## Update

Each time you publish a new tool, add another card to `index.html` with the tool name, description, and GitHub Pages link.
